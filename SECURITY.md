# Security Policy

## Vulnerability Reporting

This is a non-commercial, hobbyist project focused on low-level systems research and performance optimization. 

Contribution from everyone is well welcomed in the form of suggestions, bug reports, pull requests, and feedback.

    Reporting: If you find a security flaw or a bug, please open a GitHub Issue.

    Expectations: This project is maintained on a "best-effort" basis. There are no guaranteed response times or Service Level Agreements (SLAs).
